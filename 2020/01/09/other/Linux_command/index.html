ps aux |　grep “reg” 查看进程后通过正则表达式筛选

pwdx ID 根据进程ID查找文件路径

whatis [] 查看命令的简单描述

apropos [] 查看相关指令的文档位置

su “username” 切换用户(su默认切换root用户)

history 查看命令行记录
只能查看当前用户的命令行记录, 并且该命令无法查看文件中执行的命令行

Ctrl+l　清屏

ZZ(upper) 保存并退出VI

多文件vim

vim file1 file2   # 编辑file1和file2, 默认显示file1
:buffers          # 显示所有正在编辑的文件
:bnum             # 显示num编号的文件(num由:buffers调用后显示)
:n/N                 # 切换到下一个/上一个文件
