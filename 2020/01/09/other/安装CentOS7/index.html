other/安装CentOS7

Posted on 2020-01-09

第一次安装CentOS7的时候也是被一些问题弄得很烦, 不了了之
既然已经开始学习Linux了, 那就再试试

官网下载CentOS7 iso, 虚拟机安装之后, 默认是命令行界面, 并且没有网络

首先解决网络的问题 

进入目录    
/etc/sysconfig/network-scripts/
目录下有多个文件 编辑 ifcfg-ensXX 文件(不同环境下可能不同XX在我的环境下是33) 
修改 noboot = no -> noboot = yes
保存并退出, 再试试是否能使用网络功能

其次是安装图形界面 

// 安装图形界面程序
yum groupinstall "GNOME Desktop" "Graphical Administration Tools"
// 更新系统的运行级别(网上是这么说的, 大概意思是设置图形界面为首启动?)    
ln -sf /lib/systemd/system/runlevel5.target /etc/systemd/system/default.target
重启计算机, 默认已经是图形界面

完成

PS: 很多问题都是第一次没有解决, 再回过头来的时候, 居然出奇的轻松…
