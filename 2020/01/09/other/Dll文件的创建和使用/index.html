Posted on 2020-01-09

#Dll文件的创建和使用
在创建时需要 __declspec(dllexport)声明为导出
在使用时需要 __declspec(dllimport)声明为导入

1.使用VS创建动态库工程, 在VS创建的默认文件中(stdafx.h)声明(也可以自己创建头文件): 

__declspec(dllexport)void func();

2.然后在对应的cpp文件中编写函数(默认是以工程名命名的cpp文件)(注意要添加所使用的标准库文件):

void func()
{
    std::cout << "dll file" << std::endl;
}

3.然后生成项目, 会分别获得一个以工程名命名的.dll文件和.lib文件

4.在使用dll文件的项目中添加.lib和.dll资源文件, 添加stdafx.h头文件(需要添加stdafx中包含的头文件)

5.在使用前添加:

__declspec(dllimport)void func(); // 注意这里是import;

然后就可以正常使用func()函数了.
